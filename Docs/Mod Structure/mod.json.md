# mod.json

This is the main mod's file. There is definding mod's name, author, version, description and path to the icon

### Definitions 

`name` - the name of your mod, should be unique;<br />
`author` - the name of mod's author;<br />
`version` - mod's current version;<br />
`description` - the description of your mod;<br />
`iconPath` - path to icon, relative path starting from root mod's folder.

#### Example

```json
{
    "name": "Example mod",
    "author": "Nikon#7777",
    "version": "0.0.1",
    "description": "This is an example mod's description!",
    "iconPath": "icon.png"
}
```
